### Prerequisites

```
Node.js
Express
MySQL
```

## Installation

Clone project
```
git clone https://github.com/fblazt/library-api.git
npm init
npm install express mysql body-parser bcrypt
```

For development
```
npm install dotenv nodemon --save-dev
```



## Running the tests

Start the server:
```
npm app.js
```
Then, check every HTTP method / action from Postman.
